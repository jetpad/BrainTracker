Reaction Time Tracker

These (unfinished) examples will test reaction speed. Future versions will save each trial and analyze/chart the results using the R statistical language.

Key Test

This tests the reaction time of simply pressing a button or clicking. You do not have to make a decision before reacting. Click with the mouse, trackpad, touch screen or space bar to start and stop the timer. A running average is displayed but no information is currently saved.
